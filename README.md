# Estada Home Assistant Integration

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-orange.svg)](https://github.com/hacs/integration)

Custom integration for Estada.

## Installation via HACS

1. Open HACS in your Home Assistant instance.
2. Go to **Integrations**.
3. Click the three-dot menu → **Custom repositories**.
4. Add the URL of this repository and select **Integration** as the category.
5. Click **Add**.
6. Search for **Estada** and install it.
7. Restart Home Assistant.
8. Go to **Settings → Devices & services → Add integration → Estada**.

## Development

This repository is linked into the Home Assistant Core devcontainer at:

```text
/workspaces/core/config/custom_components/estada
```

## First setup

After running the setup script, restart Home Assistant Core and add the integration:

```text
Settings → Devices & services → Add integration → Estada
```
